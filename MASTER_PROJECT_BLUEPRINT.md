# AgriCalc Indonesia

Version : 1.0

# 1. Project Identity

## Project Name

AgriCalc Indonesia

## Project Type

Agricultural Decision Support System (ADSS)

## Project Category

Interactive Agricultural Web Platform

## Project Goal

Membangun platform digital pertanian yang membantu pengguna melakukan perhitungan, pengambilan keputusan, dan mendapatkan rekomendasi berbasis data nyata di lapangan.

Website tidak hanya berfungsi sebagai kalkulator, tetapi sebagai sistem pendukung keputusan pertanian.

---

# 2. Main Objectives

Mengubah proses perhitungan pertanian yang selama ini dilakukan secara manual menjadi sistem digital yang:

- Cepat
- Interaktif
- Informatif
- Berbasis data
- Mudah dipahami
- Mudah dikembangkan
